## yml配置文件



```yaml
#定义对象/Map
user:
 name: 我
 age: 99

#定义数组/List/Set集合
hobby:
 - java
 - game
  
```

* 缩进对齐表示层级结构
* -和:后面都有空格

* '0123456'防止误识别八进制
